# 功能特性说明

## 🎯 核心功能

### 1. 双模式连接支持

#### 🔵 Web Bluetooth API
- **支持的设备类型**
  - HM-10 / HM-11 (BLE 4.0+)
  - HC-08 (BLE)
  - MLT-BT05 (BLE)
  - 其他符合标准的BLE串口模块

- **优势**
  - ✅ 无需驱动程序
  - ✅ 真正的无线调试
  - ✅ 低功耗
  - ✅ 移动端也可支持

- **限制**
  - ⚠️ 仅支持BLE设备
  - ⚠️ 经典蓝牙(SPP)支持有限

#### 🟢 Web Serial API  
- **支持的连接方式**
  - USB串口（最常用）
  - USB转TTL
  - 虚拟串口

- **优势**
  - ✅ 稳定可靠
  - ✅ 兼容所有Arduino
  - ✅ 高速传输
  - ✅ 无需额外硬件

- **波特率支持**
  - 9600 (默认)
  - 19200
  - 38400
  - 57600
  - 115200
  - 128000 (用于虚拟示波器)

### 2. 实时数据可视化

#### 📊 三组独立图表
- **角度图表**
  - 显示平衡倾角变化
  - 用于观察平衡控制效果
  - 可看出震荡和超调情况

- **速度图表**
  - 同时显示左右轮速度
  - 对比两轮速度差异
  - 分析转向和直线运动

- **电压图表**
  - 实时监控电池电压
  - 低电量预警（黄色/红色）
  - 帮助判断性能下降原因

#### 📈 图表特性
- 平滑曲线显示
- 自动滚动更新
- 可配置数据点数（默认100点）
- 零延迟渲染
- 响应式设计

### 3. PID参数调试

#### ⚙️ 完整的PID工作流

**平衡环（Balance PID）**
- Kp: 比例系数（控制平衡力度）
- Kd: 微分系数（抑制震荡）

**速度环（Velocity PID）**
- Kp: 比例系数（控制速度响应）
- Ki: 积分系数（消除稳态误差）

#### 🔄 三步调试流程

1. **获取（Get）**
   ```
   点击 "获取PID" 
   → Arduino发送当前参数
   → 自动填充到输入框
   ```

2. **调整（Tune）**
   ```
   在输入框修改参数
   → 实时观察图表变化
   → 找到最佳参数组合
   ```

3. **保存（Save）**
   ```
   点击 "发送PID" → 应用新参数
   点击 "保存到EEPROM" → 永久保存
   ```

#### 💾 EEPROM持久化
- 参数断电不丢失
- 下次开机自动加载
- 支持参数备份恢复

### 4. 运动控制系统

#### 🕹️ 五向控制
- ⬆️ **前进** - 0x01命令
- ⬇️ **后退** - 0x05命令  
- ⬅️ **左转** - 0x06命令
- ➡️ **右转** - 0x02命令
- ⏹️ **停止** - 0x00命令

#### 🎮 控制特点
- 直观的十字按键布局
- 即按即发送
- 实时状态反馈
- 防止误操作

### 5. 智能协议解析

#### 📦 支持多种数据格式

**格式A - 传感器数据**
```
{A<左速>:<右速>:<电压>:<角度>}$
示例: {A25:28:65:2}$
```

**格式B - 传感器数据（备用）**
```
{B<角度>:<电压>:<左速>:<右速>}$
示例: {B2:65:25:28}$
```

**格式C - PID参数**
```
{C<Kp*100>:<Kd*100>:<VKp*100>:<VKi*100>}$
示例: {C1500:40:200:1}$
```

#### 🔍 智能缓冲解析
- 自动拼接分片数据
- 容错处理
- 校验数据完整性
- 识别未知格式并提示

### 6. 高级日志系统

#### 📝 分级日志显示

- **🟢 INFO** - 普通信息（灰色）
- **✅ SUCCESS** - 成功操作（绿色）
- **⚠️ WARNING** - 警告信息（黄色）
- **❌ ERROR** - 错误信息（红色）
- **🔍 DEBUG** - 调试信息（蓝色）

#### 🔧 调试功能
- 显示发送的十六进制数据
- 显示接收的原始数据
- 时间戳精确到毫秒
- 自动滚动到最新日志
- 限制行数防止卡顿

### 7. 实时统计信息

#### 📊 三项关键指标

- **接收数据包数**
  - 统计成功解析的数据包
  - 反映通信质量

- **错误计数**
  - 统计解析失败的包
  - 帮助发现协议问题

- **数据速率**
  - 实时计算pkt/s
  - 评估通信性能

### 8. 传感器数据面板

#### 📱 四项实时显示

- **平衡角度**
  - 大字体显示当前角度
  - 单位：度（°）

- **电池电压**
  - 实时电压监控
  - 三色预警系统
    - 绿色：>11.0V（正常）
    - 黄色：10.5-11.0V（注意）
    - 红色：<10.5V（警告）

- **左轮速度**
  - 编码器脉冲计数
  - 反映左轮转速

- **右轮速度**
  - 编码器脉冲计数
  - 反映右轮转速
